SMTP / Email — настройка почтовых уведомлений
Система использует SMTP для отправки уведомлений сотрудникам и менеджеру. Заполните данные вашего почтового сервера. Обычно используется порт 587 (STARTTLS) или 465 (SSL/TLS).
Пример: Портфолио IS1C <noreply@company.ru>

Управление должностями −

Список должностей, доступных для выбора сотрудниками в форме.

Аналоги должностей для выгрузки −

Задайте краткие/английские аналоги для должностей. При включённом чекбоксе в резюме (Word/PDF) будет подставляться аналог вместо полного названия.

Компетенции по должностям −

Настройте компетенции для каждой группы. Сотрудники увидят все группы в чек-листе.

Смена пароля −

Изменение пароля для вашей учётной записи ().

Управление менеджерами −

Список менеджеров, имеющих доступ к системе. Каждый входит по своему email и паролю.

Импорт сотрудников из Excel −

Импорт добавит новых сотрудников из файла к текущему списку. Существующие сотрудники (по совпадению ФИО или email) будут обновлены, остальные — сохранены без изменений.

Шаблон резюме −

Загрузите DOCX-файл с тегами {name}, {position}, {about}, {contacts}, {competencies}, {experience}, {project_experience}, {education}, {certification} в нужных местах. Данные сотрудника будут подставлены автоматически.

Обратная связь сотрудников −

Отзывы, оставленные сотрудниками через анкеты обратной связи. Собираются в таблицу вместо отправки на почту.

О системе −

IS1C Портфолио — система автоматизации данных сотрудников.

Каждый сотрудник получает уникальную ссылку для обновления своего профиля. Изменения отправляются менеджеру на проверку перед сохранением.

При наличии файла templates/resume_template.docx — резюме будет генерироваться по нему.
